Aceasta este o implementare simplificata a jocului clasic *Pac-Man*, realizata in Python folosind biblioteca **Pygame**

Proiectul include:
- labirint ( format din pereti si puncte albe)
- jucatorul - care se controleaza cu ajutorul tastelor
- fantome care se deplaseaza aleator
- ecran de GAME OVER si WIN
- posibilitatea de restart cu tasta R


  Pac- Man este controlat folosing sagetile:
  -stanga
  -dreapta
  -sus
  -jos

  Scopul jocului: sa mananci *toate punctele albe* din labirint
  Daca atingi o fantoma -> GAME OVER
  Daca reusesti sa mananci toate punctele fara a fi atins de o fantoma-> WIN
  Repornirea jocului se poate face apasand tasta **R**
